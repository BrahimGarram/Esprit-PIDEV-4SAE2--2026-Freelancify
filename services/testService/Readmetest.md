# Freelance Test Management System

## Description
Cette application Spring Boot permet de gérer des tests pour les freelances afin d'évaluer leurs compétences et déterminer s'ils sont qualifiés pour un projet ou une mission spécifique.

Fonctionnalités principales :
- Création et gestion des tests
- Ajout de questions et réponses
- Attribution des tests aux freelances
- Évaluation automatique ou manuelle des résultats
- Consultation des résultats par les responsables
